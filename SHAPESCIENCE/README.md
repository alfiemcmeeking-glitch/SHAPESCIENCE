# SHAPE SCIENCE

Experimental material design — where biology meets computation.

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Connect this repo at [vercel.com](https://vercel.com)
2. Framework preset: **Vite**
3. Build command: `npm run build`
4. Output directory: `dist`
5. Add custom domain: `shapescience.org`

## Stack

- React 18
- Vite 5
- No external UI libraries — custom design system
